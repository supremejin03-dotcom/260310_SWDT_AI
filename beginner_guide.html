📖 CHAPTER 1

바이브코딩(Vibe Coding)이란?

코드 한 줄 몰라도 됩니다. 말로 설명하면 AI가 알아서 만들어 줍니다.

💡 한마디 정의
바이브코딩(Vibe Coding)이란, 프로그래밍 지식 없이 AI에게 자연어(말/글)로 원하는 것을 설명하면 AI가 코드를 대신 작성해 주는 개발 방식입니다.

2025년 AI 연구자 안드레 카르파티(Andrej Karpathy)가 처음 사용한 용어로, 현재 전 세계 개발자·비개발자 모두 활발하게 사용 중입니다.

이런 분께 딱입니다

  • ✅ 개발을 배워보고 싶은데 코딩이 어렵다
  • ✅ 아이디어는 있는데 구현을 못한다
  • ✅ 간단한 웹사이트·자동화 도구를 만들고 싶다
  • ✅ 개발자지만 반복 작업을 빨리 처리하고 싶다
⚡

실제로 뭘 만들 수 있나요?

  • 🌐 랜딩페이지 · 포트폴리오 사이트
  • 📋 To-do 앱 · 메모 앱
  • 📊 엑셀 데이터 자동 분류 도구
  • 🤖 카카오톡 알림 자동화
  • 📱 간단한 모바일 웹앱
  • 🔄 반복 업무 자동화 스크립트
⚠️

주의할 점

  • ❗ AI가 가끔 틀린 코드를 만든다
  • ❗ 복잡한 대형 서비스는 한계가 있다
  • ❗ AI에게 DB 직접 조작은 위험
  • ❗ 보안 코드는 사람이 꼭 검수해야 한다
⚙️ CHAPTER 2

AI 코딩 도구는 어떻게 작동하나요?

크게 두 가지 방식으로 나뉩니다. 채팅형 vs CLI(터미널)형

💬 채팅형 (Chat-based)
웹 브라우저에서 대화하듯 코드를 요청하는 방식
나: "로그인 페이지 HTML 만들어줘"

AI: "네! 여기 코드입니다..."
<html>...</html>
쉬움 예: Claude.ai, ChatGPT
🖥️ CLI형 (에이전트 방식)
터미널에서 AI가 파일을 직접 읽고·쓰고·실행하는 방식
나: "쇼핑몰 장바구니 기능 추가해줘"

AI: [파일 읽는 중...]
AI: [코드 수정 중...]
AI: [테스트 실행 중...] ✅
강력함 예: Claude Code, Cursor

컨텍스트(Context)가 핵심입니다

AI에게 단순히 질문을 잘하는 것(프롬프트 엔지니어링)보다, 어떤 파일과 정보를 보여줄지 설계하는 것(컨텍스트 엔지니어링)이 더 중요합니다. Claude Code 같은 에이전트는 프로젝트 전체를 들여다보고, 관련된 파일만 골라서 읽고 수정합니다. 마치 처음부터 프로젝트에 합류한 개발자처럼요.

실사용자들의 비용 절감 꿀팁

① 플랜·설계는 Claude(고품질)로, 반복 실행은 GLM·Gemini(저비용)로 분리해서 사용
② CLAUDE.md 파일에 프로젝트 규칙을 미리 명시하면 매번 같은 설명을 반복 안 해도 됨
③ 새 세션 시작 전에 컨텍스트를 정리·압축하면 토큰 소모를 크게 줄일 수 있음
④ Gemini CLI는 무료지만 컨텍스트 창이 매우 커서 입문용으로 훌륭함

⚡

토큰이란?

AI가 처리하는 텍스트의 단위. 한글 1자 ≈ 1~2 토큰. 대화가 길어질수록, 파일이 많을수록 토큰을 더 씁니다.

🪣

컨텍스트 창

AI가 한 번에 기억할 수 있는 분량. Claude는 200K 토큰(약 15만 단어)으로 큰 편입니다.

🧩

MCP란?

Model Context Protocol. AI 도구에게 노션, GitHub 등 외부 서비스 접근 권한을 부여하는 표준 규격입니다.

📋

스킬(Skill)이란?

AI가 특정 작업(리뷰, 커밋 등)을 할 때 미리 불러오는 설명서. Claude가 오픈 표준으로 공개해 확산 중.

어디서부터 어떻게 시작할까?

실제 커뮤니티에서 검증된 입문 경로입니다. 순서대로 따라가세요.

1
🟢 STEP 1 · 1~3일 · 완전 무료

ChatGPT로 코딩 감잡기

  • 🌐
    ChatGPT.com 접속 — 무료 계정 만들기
    "간단한 HTML 페이지 만들어줘" 같은 요청으로 시작
  • 💬
    자연어로 요청 연습 — 구체적일수록 좋은 결과
    ❌ "앱 만들어줘" → ✅ "사용자가 메모를 저장할 수 있는 웹 페이지 만들어줘"
  • 📋
    결과물 직접 실행해보기 — CodePen, Replit 등 무료 실행 환경 활용
2
🔵 STEP 2 · 1~2주 · 무료

Cursor 설치 + 첫 프로젝트 완성

  • ⬇️
    cursor.com 에서 무료 설치
    VS Code 기반이라 익숙한 화면. 무료 플랜으로 충분히 시작 가능
  • 📁
    폴더 열고 "무엇을 만들고 싶은지" 입력
    Ctrl+K 로 코드 수정, Ctrl+L 로 채팅 → AI가 파일 전체를 보며 작업
  • 🎯
    목표: 간단한 To-do 앱 or 랜딩페이지 완성
    실제로 동작하는 무언가를 만들어보는 게 핵심
3
🟣 STEP 3 · 2~4주

기초 개념 병행 학습

  • 📺
    유튜브 무료 강의 활용
    "바이브코딩", "Claude Code 입문", "Cursor 튜토리얼" 검색
  • 🌐
    HTML/CSS 기초만 파악 (완벽 이해 필요 없음)
    AI가 만든 코드가 무슨 구조인지 눈으로 읽을 정도면 충분
  • 🗃️
    Supabase 로 DB 연결 실습
    무료 PostgreSQL 제공. 커뮤니티에서 가장 추천하는 초보용 DB
  • 🚀
    Vercel / Netlify 로 배포 실습
    무료로 인터넷에 내 사이트 올리기. 클릭 몇 번이면 가능
4
🟠 STEP 4 · 1~2개월

Claude Code 또는 Gemini CLI 도전

  • 💎
    Gemini CLI 먼저 시도 (완전 무료)
    Google 계정만 있으면 터미널에서 바로 사용 가능. 컨텍스트 창이 커서 큰 프로젝트도 가능
  • 🖥️
    터미널(Terminal) 기초 익히기
    cd, ls, npm, python 같은 기본 명령어 10개만 알면 충분. 나머지는 AI가 알아서
  • 📄
    CLAUDE.md 파일 작성법 배우기
    프로젝트 규칙을 AI에게 미리 알려주는 파일. 토큰 절약 + 품질 향상에 핵심
5
🌸 STEP 5 · 진행형

나만의 서비스 만들기 + 커뮤니티 참여

  • 💡
    실생활 문제를 해결하는 프로젝트 도전
    업무 자동화, 개인 프로젝트, 사이드 비즈니스 등. 작게 시작해서 키우기
  • 👥
    바이브코딩 커뮤니티 활동
    코드팩토리 카카오오픈채팅, X(트위터), 디스코드 등에서 최신 정보 실시간 공유
  • 📚
    책/강의 선택적 활용
    코드팩토리 《안티그래비티 완벽 가이드》, 인프런 강의 등 — 막히는 개념만 찾아보기
  • 🔄
    MCP 연동으로 확장
    노션, GitHub, 슬랙 등과 연결해서 AI가 더 많은 일을 하도록 권한 부여
